계정·데이터 삭제 안내

서비스명: 미세먼지 알리미 (Android)
최종 갱신: 2026년 5월 9일
본 페이지는 미세먼지 알리미 앱의 이용자가 계정 또는 서버에 보관된 개인정보의 삭제를 요청하는 절차를 안내합니다. 「개인정보 보호법」 제36조(개인정보의 정정·삭제)에 따른 권리 행사 절차이며, Google Play 정책상 앱 미설치 상태에서도 삭제를 요청할 수 있도록 본 웹 페이지를 제공합니다.

1. 본 앱의 데이터 구조 안내

먼저 데이터가 어디에 보관되는지 이해해 주세요.

데이터보관 위치삭제 방법
설정값·즐겨찾기·복약 기록·일기·운세 사주 이용자 단말 (Room DB / DataStore) 앱 삭제 → OS가 자동 제거
FCM 토큰·단말 정보 본 서비스 서버 (Supabase, 한국 서울 리전) 아래 양식으로 요청
토스 간편 로그인 표시명(이름) 이용자 단말 + 본 서비스 서버 아래 양식 + 토스 측 연결 해제
오류·충돌 로그 Firebase Crashlytics (Google) 아래 양식으로 요청
LLM 호출 메타 (호출 시각·모델·토큰 수) 본 서비스 서버 (30일 자동 삭제) 30일 내 자동 파기 / 즉시 삭제는 아래 양식

2. 가장 빠른 삭제 — 앱 내 메뉴 (앱 설치된 경우)

  1. 미세먼지 알리미 앱 실행
  2. 설정 > 개인정보 관리 > 데이터 초기화 메뉴 선택
  3. "모든 데이터 삭제" 확인 → 단말 데이터 + 서버 데이터 동시 삭제

이 방법은 즉시 처리되며 별도 신청·대기 시간이 없습니다.

3. 앱이 설치되지 않은 경우 — 이메일 요청

앱을 이미 삭제하셨거나 단말이 분실/파손된 경우, 아래 양식대로 이메일을 보내주시면 운영자가 서버 측 데이터를 직접 삭제합니다.

4. 처리 기간

  • 접수 후 10일 이내 처리·결과 통지 (「개인정보 보호법」 제36조)
  • 서버 데이터는 즉시 비활성 마킹 후 30일 이내 hard delete (이의 제기·복구 요청 대비 grace period)
  • 이용자가 즉시 hard delete 를 명시 요청하시면 grace period 없이 즉시 영구 삭제

5. 토스 간편 로그인 연결 해제

토스 측 연결 해제는 본 서비스에서 자동으로 처리되지 않습니다. 토스 측 데이터까지 함께 정리하시려면 아래 경로를 직접 실행해 주세요.

  • 토스 앱 > 전체 > 설정 > 토스 인증/간편로그인 관리 → 미세먼지 알리미 연결 해제

본 서비스 측 데이터 삭제는 위 1·2·3절 절차로 별도 진행이 필요합니다.

6. 삭제되는 데이터 / 보존되는 데이터

법령상 보존 의무가 있는 일부 기록은 삭제 요청에도 불구하고 보존됩니다.

  • ✅ 즉시 삭제: FCM 토큰, 단말 정보, 설정값, 즐겨찾기, 복약 기록, 일기, 운세 사주 입력, LLM 호출 메타, 토스 표시명
  • ⚠ 법령상 보존 (요청과 무관하게 일정 기간 유지): 마케팅 동의·철회 기록 (분쟁 대응 자체 정책 3년), 오류 로그 (Crashlytics 정책 90일)

7. 문의

삭제 절차에 어려움이 있거나 처리 결과에 이의가 있는 경우 운영자에게 직접 문의하시거나 개인정보 침해신고센터(국번 없이 118)에 연락하실 수 있습니다.
