개인정보 처리방침

서비스명: 미세먼지 알리미 (Android)
최종 개정일: 2026년 5월 9일 · 시행일: 2026년 5월 16일
미세먼지 알리미(이하 "앱")는 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 및 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」을 비롯한 관계 법령을 준수합니다. 본 방침은 앱이 어떤 정보를 수집·이용하는지, 이용자의 권리가 무엇인지 투명하게 안내합니다.

서비스 제공 범위: 본 서비스는 대한민국 거주자 대상이며, EEA(유럽경제지역)·UK·스위스 거주자에게 의도적으로 제공되지 않습니다 (not directed to EEA/UK/Switzerland).

1. 개인정보의 처리 목적

이용자는 앱 최초 실행 시 필수 항목과 선택 항목으로 구분된 동의 화면에서 각 항목을 개별적으로 선택할 수 있습니다. 앱은 아래 목적으로 이용자의 개인정보를 처리하며, 이 목적이 변경되는 경우에는 「개인정보 보호법」 제15조 제1항에 따라 별도의 동의를 받는 등 필요한 조치를 이행합니다.

  • 서비스 제공: 이용자의 현재 위치 또는 선택한 장소의 미세먼지·초미세먼지 정보를 제공하고, 대기질 등급 변동 시 푸시 알림을 발송
  • 맞춤형 설정: 연령대·장소 유형·알림 기준 등 이용자 설정을 저장하여 개인화된 알림 메시지를 제공
  • 본인확인·인증 (선택 사용 시): 토스 간편 로그인 사용자에 한해 표시명(이름) 저장
  • AI 답변 생성 (선택 기능 이용 시): 챗봇 폴백 응답·운세 심층 답변 등을 외부 LLM API 호출로 생성
  • 서비스 품질 개선: 앱 사용 통계·오류 로그 수집(동의 시)을 통해 안정성과 사용성 향상
  • 광고성 정보 발송 (선택 동의 시): 이벤트·프로모션 안내
  • 법령상 의무 이행: 관계 법령 준수, 분쟁 해결, 문의 응대

앱은 토스 간편 로그인 미사용 시 이름·전화번호·주민등록번호·계좌번호 등 직접 식별이 가능한 개인정보를 수집하지 않습니다. 토스 간편 로그인을 사용하는 경우에는 표시명 용도로 이름(user_name)만을 수집하며, 토스가 제공하는 다른 항목(이메일·전화번호·생년월일·성별·내외국인 정보·연계정보)은 토스 등록 단계에서 "사용 안함"으로 설정하여 본 서비스로 전송되지 않습니다. 또한 광고 식별자(AAID)는 현재 수집하지 않으며, 향후 광고 기능이 추가될 경우 본 방침을 개정하고 별도 동의를 받습니다.

3. 개인정보의 처리 및 보유기간

앱은 처리 목적이 달성되면 지체 없이 해당 개인정보를 파기합니다.

  • GPS 좌표: 앱 구동 중 대기질 조회 목적으로만 일시 처리, 서버에 저장하지 않음
  • 행정구역·즐겨찾기 장소: 이용자가 앱을 삭제하거나 장소를 삭제할 때까지 (단말 내부 DB)
  • FCM 토큰·단말 정보·설정값: 앱 삭제, 토큰 갱신, 또는 30일 이상 비활성 시 서버에서 자동 삭제
  • 오류·충돌 로그: 수집 시점으로부터 90일 (Firebase Crashlytics 기본 정책)
  • 마케팅 수신 동의 기록: 동의 철회 후 3년 (분쟁 대응, 「전자상거래법」 제6조)
  • 토스 간편 로그인 표시명(이름): 이용자 탈퇴 또는 토스 연결 해제 시까지. 단말 내 DataStore 저장
  • 챗봇 입력 텍스트: 30분 미상호작용 시 자동 초기화 (단말 내 메모리). LLM API 전송 시 수탁자가 학습에는 사용하지 않으나, 어뷰즈·정책 위반 모니터링 목적으로 최대 30일간 보관 후 자동 삭제 (제5-2조 참조)
  • 운세 사주 입력(이름·생년월일·시진): 이용자가 직접 삭제하거나 앱을 삭제할 때까지 단말 내 DataStore 저장. LLM 심층 답변 호출 시 수탁자가 학습에는 사용하지 않으나, Trust & Safety 목적으로 최대 30일간 보관 후 자동 삭제 (제5-2조 참조)
  • LLM 호출 메타(호출 시각·모델·토큰 수·캐시 hit 여부·차단 해시): 수집 시점으로부터 30일 (비용 통제 및 이상 탐지 용도, 호출 내용 본문은 본 서비스 측에 미저장)

4. 개인정보의 제3자 제공

앱은 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 아래의 경우 예외로 합니다.

  • 이용자의 별도 동의를 받은 경우
  • 법령상 의무이거나, 수사기관이 법령에 따라 적법한 절차에 따라 요구하는 경우

토스 간편 로그인 사용 시 비바리퍼블리카 주식회사(토스)로부터 이용자 이름(user_name)을 제공받는 흐름은 「전자서명법」상 본인확인 절차에 따른 것이며, 이는 제3자 제공이 아니라 이용자의 동의에 기반한 본인확인 사업자로부터의 정보 제공입니다.

5-3. 국외 이전 거부권 행사방법

이용자는 본 처리방침 제5-2조의 국외 이전에 대해 거부할 수 있습니다.

  • FCM·Crashlytics 거부: 앱 푸시 알림을 비활성화하면 FCM 토큰 전송이 중단됩니다. 다만 이 경우 미세먼지 등급 변동 알림 등 핵심 기능 이용이 제한됩니다.
  • AI 기능(Google Vertex AI / Anthropic) 거부: 앱 내 설정 > AI 기능 사용 토글을 OFF로 설정하면 챗봇 폴백 LLM 호출 및 운세 심층 답변이 비활성화되며, 모든 답변이 정적 콘텐츠로 대체됩니다. 이 경우 핵심 기능에는 제한이 없습니다.
  • 토스 거부: 토스 간편 로그인을 사용하지 않으면 됩니다. 익명 상태로 모든 기능을 이용할 수 있습니다. 토스 측 연결 해제는 토스 앱 > 전체 > 설정 > 토스 인증/간편로그인 관리 메뉴에서 직접 수행하실 수 있습니다.
  • Cloudflare 거부: 운영자에게 이메일(joyss00000@gmail.com)로 요청 시, 차후 우회 경로를 마련하기 전까지는 서비스 이용이 제한될 수 있습니다.

8. 개인정보 파기 절차 및 방법

앱은 보유기간이 경과하거나 처리 목적이 달성된 개인정보는 지체 없이 파기합니다.

  • 앱 내부 데이터(Room DB, DataStore): 앱 삭제 시 OS가 자동 제거

9. 개인정보의 안전성 확보 조치

12. AI 답변에 관한 안내 및 의료 가드

앱은 챗봇 폴백 응답·운세 심층 답변 등 일부 기능에서 외부 LLM API(Google Vertex AI / Anthropic) 를 호출하여 AI 답변을 생성합니다. 이용자의 권리 보호를 위해 다음 원칙을 준수합니다.

  • AI 생성 콘텐츠 표시: 챗봇 폴백 응답과 운세 심층 답변은 AI(LLM) 가 생성한 콘텐츠임을 답변 카드에 표시하며, 사실 확인 없이 의사결정 근거로 삼지 않을 것을 권장합니다.
  • 학습 거부(Opt-out) 기본 적용: 본 서비스는 모든 LLM 수탁자에 대해 이용자 입력이 모델 학습에 사용되지 않도록 "학습 OFF" 옵션을 기본값으로 설정합니다.
  • 최소 전송 원칙: 룰 기반 매칭으로 응답 가능한 질의는 LLM 에 전송하지 않습니다. 매칭 실패 시에만 일시 전송합니다.
  • 수탁자 보관 정책: LLM 호출 본문은 수탁자가 학습에는 사용하지 않으나, 어뷰즈·정책 위반 모니터링(Trust & Safety) 목적으로 최대 30일간 보관 후 자동 삭제됩니다 (제5-2조 표 참조).
  • 의료 키워드 가드(FortuneRiskGuard): 챗봇·운세 입력 텍스트에 의료·증상·진단 관련 키워드가 포함된 경우, 자동으로 LLM 호출을 차단하고 정형화된 안내문("의료 행위는 전문가와 상담하세요")으로 대체합니다. 이로써 민감정보(건강 관련)가 LLM 수탁자에 전송되는 것을 사전 차단합니다.
  • 가드 우회 시 처리: 키워드 가드를 우회하여 의료 관련 발화가 LLM 에 전송된 경우, 수탁자의 학습 OFF·30일 모니터링 후 삭제 정책에 따라 처리되며 수탁자가 영구 보관하지 않습니다. 이용자는 가능한 한 건강 관련 구체 발화를 자제해 주시기 바랍니다.
  • 운세 콘텐츠 면책: 운세 답변(정적 풀 + LLM 심층 답변 모두)은 오락 목적이며 종교·신념·점술적 효력을 보장하지 않습니다. 모든 운세 결과에 "재미로 보는 운세" 디스클레이머가 자동 첨부됩니다. AI 답변에 관한 본 조의 안내는 운세 LLM 답변에도 동일하게 적용됩니다.
  • 14세 미만 사주 입력 차단: 운세 사주 입력 폼은 만 14세 미만에게 노출되지 않으며, 입력 단계에서 자동 차단됩니다.
  • 자동화된 의사결정 미사용: 본 서비스의 AI 답변은 이용자에게 법적 효력이나 중대한 영향을 미치는 자동화된 의사결정에 사용되지 않습니다.
  • AI 기능 거부: 이용자는 앱 내 설정 > AI 기능 사용 토글을 OFF 로 설정하여 LLM 호출 자체를 거부할 수 있습니다. 이 경우 모든 답변이 정적 콘텐츠로 대체되며, 핵심 기능에는 제한이 없습니다.
