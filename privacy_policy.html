개인정보 처리방침

서비스명: 미세먼지 알리미 (Android)
최종 개정일: 2026년 4월 27일 · 시행일: 2026년 5월 1일
미세먼지 알리미(이하 "앱")는 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」, 「위치정보의 보호 및 이용 등에 관한 법률」을 비롯한 관계 법령을 준수합니다. 본 방침은 앱이 어떤 정보를 수집·이용하는지, 이용자의 권리가 무엇이며 어떻게 행사할 수 있는지 투명하게 안내합니다.
본 사업자는 「위치정보의 보호 및 이용 등에 관한 법률 시행령」 제6조에 따라 개인위치정보를 이용자의 단말기에서만 활용할 뿐 사업자의 위치정보시스템으로 전송·저장하지 아니하므로 위치기반서비스사업 신고 면제 대상입니다. 위치정보의 처리에 관한 상세 사항은 별도의 「위치기반서비스 이용약관」을 참고하시기 바랍니다.

1. 개인정보의 처리 목적

앱은 아래 목적으로 이용자의 개인정보를 처리하며, 이 목적이 변경되는 경우에는 「개인정보 보호법」 제18조에 따라 별도의 동의를 받는 등 필요한 조치를 이행합니다.

  • 서비스 제공: 이용자의 현재 위치 또는 선택한 장소의 미세먼지·초미세먼지 정보를 제공하고, 대기질 등급 변동 시 푸시 알림을 발송
  • 맞춤형 설정: 연령대·장소 유형·알림 기준 등 이용자 설정을 저장하여 개인화된 알림 메시지를 제공
  • 서비스 품질 개선: 앱 사용 통계·오류 로그 수집(동의 시)을 통해 안정성과 사용성 향상
  • 광고성 정보 발송 (선택 동의 시): 이벤트·프로모션 안내
  • 법령상 의무 이행: 관계 법령 준수, 분쟁 해결, 문의 응대

3. 개인정보의 처리 및 보유기간

앱은 처리 목적이 달성되면 지체 없이 해당 개인정보를 파기합니다.

  • GPS 좌표: 앱 구동 중 대기질 조회 목적으로만 일시 처리, 서버에 저장하지 않음
  • 행정구역·즐겨찾기 장소: 이용자가 앱을 삭제하거나 장소를 삭제할 때까지 (단말 내부 DB)
  • FCM 토큰·단말 정보·설정값: 앱 삭제, 토큰 갱신, 또는 30일 이상 비활성 시 서버에서 자동 삭제
  • 오류·충돌 로그: 수집 시점으로부터 90일 (Firebase Crashlytics 기본 정책)
  • 마케팅 수신 동의 기록: 동의 철회 후 3년 (분쟁 대응, 「전자상거래법」 제6조)

6. 공공데이터 API 이용 안내

앱은 직접 외부 API를 호출하지 않고, 자체 서버(Supabase Edge Functions)가 아래의 공공 또는 외부 API를 호출하여 수집한 대기질·뉴스·의료기관 정보만 중계받습니다. 이 과정에서 이용자의 개인정보는 외부 API에 전송되지 않습니다. 외부 API 호출에 필요한 키는 서버 측 보안 저장소에만 보관되며 앱 배포물(APK)에는 포함되어 있지 않습니다.

  • 에어코리아 (환경부·한국환경공단) 대기오염정보 서비스 — 전국 실측 PM2.5/PM10
  • 서울특별시 열린데이터광장 대기질 정보 — 서울 자치구별 보강
  • OpenWeatherMap Air Pollution API — 전 세계 대기오염 보조 소스
  • 기상청(KMA) 공공데이터 — 기상 실황 보조 신호
  • 네이버 뉴스 검색 OpenAPI — "미세먼지" 관련 최신 뉴스 조회
  • 건강보험심사평가원(HIRA) 병원정보서비스 · 보건복지부 응급의료기관 — 근처 병원·보건소 조회

7. 정보주체의 권리·의무 및 행사방법

이용자는 「개인정보 보호법」 제35조 내지 제37조의2에 따라 언제든지 다음의 권리를 행사할 수 있습니다.

  • 열람권: 본인의 개인정보에 대한 열람 요구
  • 정정·삭제권: 사실과 다른 개인정보의 정정 또는 삭제 요구
  • 처리정지권: 개인정보 처리의 일시 또는 영구 정지 요구
  • 동의 철회권: 수집·이용·제공·국외이전 등 개별 동의에 대한 철회
  • 전송요구권: 「개인정보 보호법」 제35조의2에 따라 본인의 개인정보를 다른 개인정보처리자에게 전송할 것을 요구 (단, 앱은 현재 다른 처리자로의 직접 전송을 지원하지 않으며, 이용자가 요청 시 본인에게 사본을 제공하는 방식으로 대응)
  • 자동화된 결정에 대한 거부권·설명요구권: 본 방침 제16조 참조
  • 만 14세 미만 아동의 경우, 법정대리인이 해당 아동의 권리를 대리 행사

권리 행사 방법 (앱 내 자가서비스)

아래 메뉴에서 즉시 권리를 행사할 수 있으며, 별도의 신청·승인 절차 없이 본인이 직접 처리 가능합니다.

  • 설정 → 동의 관리: 마케팅·야간 광고 동의 ON/OFF, 위치 권한 회수 안내
  • 설정 → 내 데이터 관리 → 데이터 내보내기: 일기·복약·등록 장소를 JSON 파일로 추출
  • 설정 → 내 데이터 관리 → 내 데이터 모두 삭제: 단말 내부 모든 개인정보 영구 삭제 + 서버에 보관 중인 FCM 토큰·단말 정보 즉시 삭제 요청
  • 앱 삭제: 설치된 앱과 단말 내부 데이터 일괄 삭제

위 방법으로 처리가 어렵거나 추가적인 권리 행사가 필요한 경우, 개인정보 보호책임자 이메일 joyss00000@gmail.com으로 서면 요청하실 수 있습니다. 요청 접수 후 정당한 사유가 없는 한 10일 이내에 결과를 통지합니다.

8. 개인정보 파기 절차 및 방법

앱은 보유기간이 경과하거나 처리 목적이 달성된 개인정보는 지체 없이 파기합니다.

  • 전자적 파일: 복구·재생이 불가능한 기술적 방법으로 영구 삭제
  • FCM 토큰·단말 정보: 서버 DB에서 UPDATE 또는 DELETE를 통해 즉시 제거
  • 앱 내부 데이터(Room DB, DataStore): 앱 삭제 시 OS가 자동 제거

9. 개인정보의 안전성 확보 조치

앱은 「개인정보 보호법」 제29조에 따라 다음과 같은 안전성 확보 조치를 취하고 있습니다.

  • 기술적 조치: HTTPS/TLS 암호화 전송, Android Network Security Config 적용, 단말 내 설정은 OS 샌드박스 내 저장, 자동 백업 비활성화
  • 관리적 조치: 개인정보 접근 권한 최소화, 수탁자와 개인정보 처리계약 체결
  • 물리적 조치: 수탁 서버 운영사의 보안 정책 준수

10. 자동수집 장치(쿠키 등)의 설치·운영 및 거부

앱은 웹 브라우저에서 사용하는 쿠키를 사용하지 않습니다. 앱 내 WebView가 개인정보 처리방침 문서를 표시하는 용도로만 사용되며, 이 WebView는 JavaScript와 쿠키를 비활성화합니다.
