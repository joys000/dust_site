# ☕ 후원자 명단 자동 추가 — 사용법

macOS와 Windows 모두에서 작동합니다.

---

## 🍎 macOS 처음 한 번만

터미널 열고:
```bash
cd ~/Desktop/dustalert-site
chmod +x 명단추가.command add-donor.py
```

> Finder 더블클릭 시 "확인되지 않은 개발자" 경고가 뜨면, 우클릭 → 열기 한 번만 해주시면 다음부터는 더블클릭으로 바로 실행됩니다.

## 🪟 Windows 처음 한 번만

1. **Python 3 설치** (필수, 1회)
   - https://www.python.org/downloads/ 에서 최신 다운로드
   - 설치 시 **"Add Python to PATH"** 체크박스 꼭 켜기
2. **Git for Windows 설치** (자동 push 쓰려면)
   - https://git-scm.com/download/win 에서 다운로드
3. 끝. 별도 권한 부여 필요 없음

> 처음 `명단추가.bat` 더블클릭하면 Windows Defender SmartScreen 경고가 뜰 수 있어요.
> "추가 정보" → "실행" 한 번만 해주시면 다음부턴 바로 실행돼요.

## 📌 실제 사용 방법 (둘 다 동일)

### 방법 1 — 한 명씩 입력 (가장 단순)

1. **macOS**: `명단추가.command` 더블클릭
   **Windows**: `명단추가.bat` 더블클릭
2. `1` 선택 (한 명씩 입력)
3. 이름·메시지·날짜·채널 차례로 입력
4. 미리보기 확인 → `y`
5. GitHub 자동 push 여부 → `y`
6. 1–2분 후 사이트 반영 ✨

### 방법 2 — Sheets에서 일괄 복사·붙여넣기 (가장 빠름)

1. Form #2 응답 시트 열기
2. 새 후원자 행들 선택 → **Cmd+C** (Mac) / **Ctrl+C** (Win)
3. **`명단추가.command`** 또는 **`명단추가.bat`** 더블클릭
4. `2` 선택 (일괄 붙여넣기)
5. **Cmd+V** / **Ctrl+V** 로 붙여넣기 → Enter
6. 빈 줄 두 번 (Enter Enter) → 끝 신호
7. 미리보기 확인 → `y` → 자동 push

### 방법 3 — CSV 파일 드래그

1. Sheets → 파일 → 다운로드 → CSV
2. 다운로드한 CSV 파일을 **`명단추가.command`** 또는 **`명단추가.bat`** 위로 드래그앤드롭
3. 자동 파싱 → 미리보기 → `y` → 자동 push

## 🎯 자동 인식 항목

스크립트가 알아서 처리해줍니다:

| 입력 | 자동 변환 |
|---|---|
| `카카오페이`, `카카오`, `kakao`, `💛` | → `kakao` |
| `KB국민`, `KB`, `국민` | → `kb` |
| `토스뱅크`, `토스`, `toss` | → `toss` |
| `2026.04.27`, `2026/04/27`, `2026년 4월 27일` | → `2026-04-27` |
| 빈 날짜 | → 오늘 날짜 |
| 표시 이름이 비어 있으면 | → 송금자 이름 사용 |
| 명단 공개 미동의자 | → 자동 건너뜀 |
| 따옴표(`'`) 메시지 | → 자동 이스케이프 |

## 🔁 자동 git push 흐름

스크립트 마지막에 `y` 누르면 자동으로:
1. `git add donors-data.js`
2. `git commit -m "후원자 추가 (N명)"`
3. `git push`

→ 1–2분 후 사이트에 반영됩니다.

## ❓ 자주 묻는 문제

### macOS

**Q. 더블클릭하면 "권한 없음" 에러**
A. `chmod +x 명단추가.command` 한 번 실행하셨는지 확인하세요.

**Q. "확인되지 않은 개발자" 경고**
A. Finder에서 우클릭 → 열기. 한 번만 해두면 다음부턴 더블클릭으로 OK.

**Q. Python 3가 없다고 나와요**
A. 메시지 안내대로 `xcode-select --install` 실행하시면 자동 설치됩니다.

### Windows

**Q. 한글이 깨져 보여요 (■■■ 같이)**
A. 콘솔 폰트가 한글 미지원. `명단추가.bat` 안의 `chcp 65001` 명령이 자동 실행되긴 하지만, 그래도 깨지면 cmd 우클릭 → 속성 → 폰트 → "맑은 고딕"이나 "굴림체"로 변경.

**Q. "python is not recognized" 에러**
A. Python 설치 시 **"Add Python to PATH"** 체크 안 하셨어요. Python 재설치 후 그 옵션 체크해주세요.

**Q. SmartScreen 경고**
A. "추가 정보" → "실행" 한 번만 누르시면 됩니다.

**Q. git push가 안 돼요**
A. Git for Windows 설치 + GitHub 인증 설정 필요. 처음이라면 [GitHub Desktop](https://desktop.github.com)을 쓰시는 게 더 편할 수도 있어요. 그 경우 자동 push는 `n` 누르고 GitHub Desktop으로 commit/push.

### 공통

**Q. 잘못 추가했어요**
A. `donors-data.js` 파일 열고 그 줄 직접 지우시면 됩니다. 그 다음 git push.

**Q. 일괄 모드에서 일부만 추가하고 싶어요**
A. 미리보기 단계에서 `n` 누르면 취소됩니다. Sheets에서 원하는 행만 선택 후 다시 시도하시면 돼요.

**Q. 폼 응답 컬럼 순서가 달라요**
A. 스크립트가 4/5/6/7/8 컬럼 자동 감지합니다. 그래도 안 되면 직접 모드 1번(한 명씩 입력) 사용하시거나, donors-data.js를 직접 편집하세요.
